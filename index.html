❓ How to Use SmartLab

Step 1 — Choose Connection

Select the connection type from the Hardware Connection panel:

  • 🔌 USB / Serial: Connect Arduino/ESP32 via USB cable, then select "Web Serial" and click "Start Test" — the browser will ask you to pick the port.
  • 🌐 LAN / Network: Enter the device IP address (e.g. 192.168.1.50). The device must run a WebSocket server on the SmartLab protocol.
  • 📡 Bluetooth: Click "Scan & Pair Bluetooth" to find nearby BLE devices running the SmartLab firmware.
  • ✏️ Manual Mode: Enter moisture % and force (N) values manually for each strike — no hardware needed.
  • 🧪 Demo Mode: Simulates sensor data for testing the interface.

Step 2 — Set Test Parameters

Enter Hammer Weight, Mold Volume, Target Strikes, and Target Ratio in the Engineer Inputs panel.

Step 3 — Start & Collect Data

Click Start Test. Each hammer strike will be recorded automatically (or manually). Watch the live gauge, strike log, and density calculations update in real-time.

Step 4 — Results & Report

After the target strikes, results are calculated automatically. Click Download PDF to generate a professional lab report.

🛠️ Supported Hardware & Features

Supported Sensors

⚖️ HX711 Load Cell

High-precision weight/force sensor (24-bit ADC). Used for measuring compaction force, CBR load, Marshall stability. resolution: ±0.1 N.

💧 Capacitive Soil Moisture Sensor

Capacitive v1.2 / v2.0 — corrosion-resistant soil moisture measurement. Range: 0–100%. Analog output, calibrated via SmartLab firmware.

📡 Ultrasonic HC-SR04

Distance measurement for digital slump test and settlement monitoring. Range: 2cm–400cm, accuracy: ±3mm.

🧭 MPU6050 IMU (Gyroscope + Accelerometer)

6-axis motion sensor for road roughness measurement and inclinometer applications. I2C interface.

🌡️ DS18B20 Temperature Sensor

Digital temperature sensor for concrete maturity monitoring. ±0.5°C accuracy. OneWire protocol, multiple sensors on single bus.

📏 LVDT Displacement Sensor

Linear Variable Differential Transformer for precise displacement/flow measurement in Marshall test. Range: ±25mm.

Platform Features

  • Real-time sensor streaming via Web Serial / WebSocket / Bluetooth
  • Automatic calculation of density, compaction ratio, and engineering formulas
  • Strike-by-strike logging with live gauges
  • PDF report generation with jsPDF
  • Cloud session history via Firebase Firestore
  • AI Engineering Team (5 agents) for instant technical support
  • 7-language support (EN, AR, ZH, DE, FR, JA, RU)

📱 QR Specimen Tracking

Track specimens from casting to testing with QR codes — full chain of custody.

📋 Workflow

1Cast — Generate QR at casting, attach to specimen
2Cure — Scan QR when placing in curing tank
3Test — Scan QR at compression machine, auto-load data
4Report — Auto-generate report with full traceability

🏷️ Specimen Data Captured

  • Project name & location (GPS)
  • Mix design & truck ticket number
  • Casting date/time & technician ID
  • Fresh properties (slump, air, temperature)
  • Curing conditions & tank number
  • Break date, load, strength, break type

💡 Coming in v2.0

QR generation, Bluetooth label printing, and mobile scanning will be available in the next release. Currently, specimens can be manually tracked through the session history.

📅 Break Schedule Generator

Auto-generate daily break lists from casting dates. Never miss a 7-day or 28-day break.

⚙️ How It Works

  1. Enter casting date for each batch
  2. System calculates all break dates automatically
  3. Daily list shows which specimens are due today
  4. Color-coded: 🟢 On time / 🟡 Approaching / 🔴 Overdue
  5. Export as PDF for lab technicians

💡 In v2.0, this will integrate with session history to auto-track specimens and generate printable daily break lists.
